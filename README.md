# 🛰️ Packet Sniffer / Network Traffic Analyzer

## 📌 Description

This project is a simple Packet Sniffer built using Python and Scapy. It captures and analyzes live network traffic, displaying useful information such as source IP, destination IP, protocol type, and timestamps.

---

## 🚀 Features

* Capture live network packets
* Display Source and Destination IP addresses
* Identify protocols (TCP, UDP, ICMP)
* Packet counting
* Save captured data to a log file
* Timestamp for each packet

---

## 🛠️ Technologies Used

* Python 3
* Scapy Library
* Kali Linux

---

## ▶️ How to Run

1. Install dependencies:

```bash
sudo apt install python3-scapy
```

2. Run the program:

```bash
sudo python3 sniffer.py
```
